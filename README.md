# Bowling-League-Score-Tracker


A high-logic Spring Boot application designed to manage competitive bowling leagues, track frame-by-frame scores, and generate real-time performance analytics.🎯 Project OverviewUnlike a standard CRUD app, this system implements the complex Non-Linear Scoring Algorithm of bowling (handling Strikes and Spares which depend on future rolls). It aggregates raw game data into a professional league leaderboard.Key FeaturesAutomatic Score Calculation: Logic-heavy service layer to calculate frame scores, including "tenth frame" bonus rolls.Dynamic Leaderboard: Real-time calculation of player standings and win percentages.Handicap Engine: Automated calculation of player handicaps based on league-standard averages.Match History: Detailed breakdown of every game played by a specific user.📊 League Standings FormatThe API calculates and returns the following dataset for the frontend dashboard:RankPlayerWinsTop ScoreWin%LossesGames Played1Alex Rivera1830090%2202Sam Smith14265
